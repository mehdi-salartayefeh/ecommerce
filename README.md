# My sample ecommerce website
a sample ecommerce site that I made using Nodejs, React, and Nestjs


### Technology Stack: 
- Front-end
    - ReactJs
    - mdb-react
    - HTML5
    - CSS3 
    - SCSS & LESS
    - State management in MyState.js
- Back-end    
    - RESTFUL (Connectivity)
    - NodeJs (Language)
    - NestJs (Framework)
    - Mongoose (ODM)
    - Mongodb (Database)
    - PM2
