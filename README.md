# My sample ecommerce website
a sample ecommerce website I made using Nodejs, React, and Nestjs

## Demo
<a href="https://noora.salartayefeh.ir/" target="_blank">See Demo</a>


### Technology Stack: 
- Front-end
    - ReactJs
    - mdb-react
    - HTML5
    - CSS3 
    - SCSS & LESS
    - State management in MyState.js
- Back-end    
    - RESTFUL (Connectivity)
    - NodeJs (Language)
    - NestJs (Framework)
    - Mongoose (ODM)
    - Mongodb (Database)
    - PM2
- Server
  - Linux CentOS
  - Caddy
